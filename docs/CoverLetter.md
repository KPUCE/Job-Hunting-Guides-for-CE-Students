# 1. 자기소개서 작성 시 일반적인 유의 사항

## 서류심사관이나 면접관들은 거의 모두 자기소개서를 통해 지원자를 파악한다
  * 가장 자신이 있는 부분을 자기소개서에 기술하여 면접 시 질문을 유도하라. 면접자는 자기소개서의 내용을 읽고 질문을 결정하는 경우가 대부분이다. 
  * 자기소개서에 기술한 내용은 정확하게 파악하고 있어야 한다. 일단 자기소개서에 기술했다면 답변이 막히면 안된다.
  * 자기소개서에 거짓말을 써서는 절대 안된다. 거짓말은 탄로나기 쉽고 탄오나면 바로 탈락이다. 자신없는 부분은 차라리 기술하지 않는 것이 낫다.
  * 개인의 인성을 파악할 내용이 아니면 개인 신상을 너무 많이 기술하지 마라. 면접자는 개인 신상에 별로 관심이 없는 경우가 많다.
