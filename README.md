# Job-Hunting-Guides-for-CE-Students

  * 인생에 있어 처음 직장을 잡는 것은 대부분의 사람에게 매우 어렵고 고통스러운 작업입니다. 어떤 사람은 5 ~ 6개의 지원을 통해 쉽게 구직하는 경우도 있지만, 어떤 사람은 100개 이상의 지원을 해도 잘 안 풀리는 경우가 있어 절망하는 경우도 있습니다. 처음에 잘 풀리지 않더라도 인내를 가지고 잘 준비해서 꾸준히 지원하다 보면 어디엔가 나에게 맞는 직장이 나타날 거라고 생각하며 과정을 즐겼으면 합니다.
  * 컴퓨터공학 졸업예정자(취업준비생)의 능력에 차이가 있으므로 현재 본인의 역량을 잘 분석하셔서 지원할 회사를 결정할 필요가 있습니다. 처음부터 본인의 마음에 드는 회사에 들어가면 좋겠지만 매우 힘든 과정이니, 눈을 조금 낮춰 입사를 하고 3년정도 다니다가 이직을 하는 방법도 좋은 전략입니다. IT 분야는 이직이 매우 용이하고, 이직할 때 연봉 등의 조건이 훨씬 좋아지는 경우가 많으니 참고하기 바랍니다. 특히 자신의 역량이 부족하다고 생각하는 학생들은 4학년 2학기에 중소기업에 들어가서 중고 신입으로 마음에 드는 회사에 지원하는 것도 추천하는 전략입니다.
  * 구직에 있어 가장 중요한 것이 본인의 역량을 높이는 것이 매우 중요하지만, 자기소개서를 잘 작성하여 본인을 잘 어필하는 것도 매우 중요한 일입니다. 아래에 자기소개서 작성 시 알아야 하는 주요 유의 사항을 안내하니, 읽어보시고 자기소개서 작성에 참고하여 좋은 결과 있기 바라겠습니다.
